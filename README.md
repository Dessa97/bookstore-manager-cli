# Bookstore Manager CLI

Uma CLI simples para gerenciar uma livraria.

## Requisitos

- Node.js 18+
- npm

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Compilar

```bash
npm run build
```

## Executar

```bash
npm start
```

## Uso como comando global

Após instalar globalmente ou criar um link simbólico, execute:

```bash
npx bookstore-manager-cli
```

## Estrutura

- `src/` - código-fonte TypeScript
- `dist/` - saída compilada
- `tsconfig.json` - configuração do TypeScript
- `package.json` - scripts e dependências
